About:
ONU OS is a Linux distribution for development oiginally based on Debian 12

Features

  • Simple: ONU is a very minimalist operating system.
  • Good development environment: Designed for efficient software creation.
  • Easy to use: A straightforward graphical interface makes it intuitive.
  • Open-source customization: You can create your own version/flavor by editing the build script on GitHub and submitting it by e-mailing me your flavor's info and its GitHub URL.

Download ONU OS

To download and build ONU, click the button below. On a Debian-based environment, run the following commands:

chmod +x build-onu-plus.sh
./build-onu-plus.sh

This will produce the file ~/ONU/live-image-amd64.hybrid.iso. You can test it live or install it via Calamares from the desktop.
